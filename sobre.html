Sobre este Projeto

Este projeto foi desenvolvido como parte das atividades da disciplina da trilha de Front-end Javascript no Módulo 03 (Front End Dinâmico (JS DOM)) do programa DiversAbility.

O objetivo é concluir o desafio do professor e demonstrar o domínio das habilidades apresentadas em aula.

Todas as seções apresentam exercícios práticos implementados de forma interativa.

Tecnologias Utilizadas

TypeScript: Linguagem principal para desenvolvimento
HTML5: Estrutura semântica das páginas
CSS3: Estilização com variáveis CSS e design responsivo
FakeStore API: API REST para operações CRUD
LocalStorage: Persistência de sessão do usuário
Vite: Build tool e dev server

Funcionalidades Implementadas

  • ✅ Sistema de autenticação de usuários
  • ✅ Listagem de produtos com paginação virtual
  • ✅ Modal de detalhes dos produtos
  • ✅ CRUD completo de produtos (Create, Read, Update, Delete)
  • ✅ Validação de formulários no front-end
  • ✅ Controle de permissões (apenas "johnd" pode deletar)
  • ✅ Design responsivo para mobile e desktop
  • ✅ Tratamento de erros e loading states
  • ✅ Persistência de sessão com localStorage
  • ✅ Interface moderna com tema dark
